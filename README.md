## Cel projektu:

Celem tego projektu jest stworzenie bazy danych, która będzie pobierać dane na temat zespołów muzycznych z Musicbrainz API w systematyczny sposób.

## Pytania biznesowe:

 **Geografia metalu per capita** — w jakim kraju per milion mieszkańców 
   jest najwięcej zespołów metalowych?

**Trend DnB per dekada** — jak rozkłada się powstawanie zespołów DnB 
   na dekady (90s, 00s, 10s, 20s)? Czy scena rośnie czy umiera?

**Długość życia zespołów: rock vs metal vs DnB** — jaka jest średnia 
   długość życia zespołu w każdej z 3 scen?

**Soliści vs zespoły w scenach rock/metal/DnB** — jaki procent 
   wykonawców to Person, a jaki Group, w każdej scenie?

**Kraje gdzie metal dominuje nad rockiem** — w których krajach jest 
   więcej zarejestrowanych zespołów metalowych niż rockowych?
